# Rules — 콘텐츠 작성 규칙

포스트 작성 시 반드시 준수. AI가 초안 작성해도 이 규칙이 기준.

## 문서 목록

| 파일 | 용도 | 언제 읽나 |
|------|------|-----------|
| `CONTENT-RULES.md` | Frontmatter · Voice & Tone (AI 냄새 12 패턴) · 본문 · 코드 블록 · 이미지 · 교차 게시 | 포스트 초안 전 필수 |

## 파일 추가 기준

- **Yes**: 모든 포스트에 적용되는 작성 규칙 (Voice, 스타일, frontmatter 필드 등)
- **No**: 특정 포스트 한정 가이드 → 해당 포스트 주석으로

## 관련 폴더

- `../state/CONTENT-PLAN.md` — 계획된 포스트 목록 (규칙 적용 대상)
- `../state/POST-INDEX.md` — 실제 게시 현황
- `../templates/` — 페이지(포스트 아님) 템플릿
